## 2. Best of Batch

- `Chat Answer Evidence Drawer`: survives because it materially improves trust in AI answers while reusing existing chart-backed context. It is stronger than novelty AI ideas because it upgrades the current assistant loop.
- `Room Provenance Reconciliation`: survives because shared-expense trust is a real product risk once receipts, statements, and manual entries mix together. It is stronger than lighter room nudges because it fixes a source-of-truth problem.
- `Limit Forecast Banner`: survives because it improves both user experience and monetization without requiring a new surface. It is stronger than generic paywall copy because it uses the user’s real projected usage.
- `Grocery Budget Guardrail`: survives because it connects Fridge to budgeting and follow-through instead of leaving grocery analysis as a passive readout.
- `SSO Resume Destination Guard`: survives because it removes a brittle auth seam on real product routes with low implementation risk.
- `Billing Return Recovery Hub`: survives because it makes a real conversion-critical system page more legible right after billing transitions.
